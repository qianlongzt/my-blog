基本句式 

{if $name eq “Fred”}
    Fred
{elseif $name eq “Wilama”}
     Wilama
{else}
    else
{/if}
条件修饰符有很多，请记得简单的几个eq ( == ) neq ( !eq ) gt ( > ) lt (< )

修饰符必须和变量或常量用空格分开
